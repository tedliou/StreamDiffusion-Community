# Platform Compatibility Checklist

- [ ] Prefer Bash command examples over PowerShell-first examples.
- [ ] Prefer POSIX paths over backslash-style paths unless explicitly required.
- [ ] Check whether Ubuntu or WSL2 wording is more appropriate than generic Windows wording.
- [ ] Confirm environment setup instructions match Linux-native tooling.
- [ ] Mark any unavoidable platform-specific exceptions clearly.
- [ ] Avoid implying support that has not been verified.
